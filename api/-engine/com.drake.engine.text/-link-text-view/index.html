使 android.widget.TextView 能自动识别 URL、电话、邮箱地址。 相比于 android.widget.TextView 使用 android.text.util.Linkify, LinkTextView 有以下特点:

  • 可以通过 LinkTextView 中的属性设置链接的样式
  • 可以通过 setOnLinkClickListener 设置链接的点击事件， 而不是 android.widget.TextView 默认的 android.content.Intent 跳转

使 android.widget.TextView 能自动识别 URL、电话、邮箱地址。 相比于 android.widget.TextView 使用 android.text.util.Linkify, LinkTextView 有以下特点:
  • 可以通过 LinkTextView 中的属性设置链接的样式
  • 可以通过 setOnLinkClickListener 设置链接的点击事件， 而不是 android.widget.TextView 默认的 android.content.Intent 跳转

Content copied to clipboard
修复了 TextView 与 android.text.style.ClickableSpan 一起使用时， 点击 android.text.style.ClickableSpan 也会触发 TextView 的事件的问题。 同时通过 setNeedForceEventToParent 控制该 TextView 的点击事件能否传递给其 Parent， 修复了 TextView 默认情况下如果添加了 android.text.style.ClickableSpan 之后就无法把点击事件传递给 TextView 的 Parent 的问题。 注意: 使用该 TextView 时, 用 TouchableSpan 代替 android.text.style.ClickableSpan, 且同时可以使用 TouchableSpan 达到修改 span 的文字颜色和背景色的目的。 注意: 使用该 TextView 时, 需调用 setMovementMethodDefault 方法设置默认的 LinkTouchMovementMethod, TextView 会在 onTouchEvent 时将事件传递给 LinkTouchMovementMethod, 然后传递给 TouchableSpan, 实现点击态的变化和点击事件的响应。
